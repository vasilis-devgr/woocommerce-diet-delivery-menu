# WooCommerce Diet Delivery Menu

A WooCommerce plugin for Online Delivery Plates Menu

# Description

Online Delivery Menu. One of its functions is to allow users to order online from specific menus and then complete their payment through the WooCommerce checkout environment. Another function is to allow the administrator to create the menus themselves, entering meals and grouping them as desired. The ACF Pro and WooCommerce plugins were used for its development.

# Installation

1. Install ACF Pro plugin to the back end.
2. Install Elementor Pro page builder
3. Upload .zip file to the back end and activate
4. Create a page and use shortcode [blue_menus]
